The Code I Stopped Writing as a Flutter Developer (And What I Use Instead)

Iqra Ilyas
3 min read
·
Sep 9, 2025

--

Flutter is world for beginners.

While started working with flutter, I wrote a lot of code that worked well but it wasn’t clean and efficient. By a time, I found different ways to make it more optimized and scalable. I just learned some better patterns to work on.

In this post, I’ll share real examples of code I stopped writing — and the better alternatives I use today. If you’re starting your Flutter journey, this might save you some headaches.

1. Manual setState Everywhere

Before: 

                                                                        
                                                                            class CounterScreen extends StatefulWidget {

                                                                            @override

                                                                            _CounterScreenState createState() =>_CounterScreenState();
}
                                                                        
                                                                    
                                                                        
                                                                            class _CounterScreenState extends State <CounterScreen >{
int counter = 0;
                                                                        
                                                                        
                                                                            void increment() {

                                                                            setState(() {

                                                                            counter++;

                                                                            });
}
                                                                        
                                                                        
                                                                            @override

                                                                            Widget build(BuildContext context) {

                                                                            return Column(

                                                                            children: [

                                                                            Text('$counter '),

                                                                            ElevatedButton(

                                                                            onPressed: increment,

                                                                            child: Text('Increment '),

                                                                            ),

                                                                            ],

                                                                            );

                                                                            }
}

After (using Provider): 

                                                                        
                                                                            class CounterProvider extends ChangeNotifier {
int counter = 0;
                                                                        
                                                                    
                                                                        
                                                                            void increment() {

                                                                            counter++;

                                                                            notifyListeners();

                                                                            }
}
                                                                        
                                                                        
                                                                            class CounterScreen extends StatelessWidget {

                                                                            @override

                                                                            Widget build(BuildContext context) {
final provider = Provider.of <CounterProvider >(context);
                                                                        
                                                                        
                                                                            return Column(

                                                                            children: [

                                                                            Text('${provider.counter}'),

                                                                            ElevatedButton(

                                                                            onPressed: provider.increment,

                                                                            child: Text('Increment '),

                                                                            ),

                                                                            ],

                                                                            );

                                                                            }
}

✅ Cleaner, scalable, and easy to reuse in multiple widgets.

2. Hardcoding Styles in Every Widget

Before: 

                                                                        
                                                                            Text(

                                                                            "Hello Flutter ",

                                                                            style: TextStyle(

                                                                            fontSize: 20,

                                                                            fontWeight: FontWeight.bold,

                                                                            color: Colors.blue,

                                                                            ),
);

After (using ThemeData + Google Fonts): 

                                                                        
                                                                            Text(

                                                                            "Hello Flutter ",

                                                                            style: Theme.of(context).textTheme.titleLarge,
);

✅ Centralized styles → easier maintenance + consistent UI.

3. Nesting Too Many Rows and Columns

Before: 

                                                                        
                                                                            Row(

                                                                            children: [

                                                                            Column(

                                                                            children: [

                                                                            Text("Name "),

                                                                            Text("Iqra "),

                                                                            ],

                                                                            ),

                                                                            SizedBox(width: 20),

                                                                            Column(

                                                                            children: [

                                                                            Text("Role "),

                                                                            Text("Flutter Dev "),

                                                                            ],

                                                                            ),

                                                                            ],
);

After (using Wrap ): 

                                                                        
                                                                            Wrap(

                                                                            spacing: 20,

                                                                            children: [

                                                                            Column(

                                                                            children: [Text("Name "), Text("Iqra ")],

                                                                            ),

                                                                            Column(

                                                                            children: [Text("Role "), Text("Flutter Dev ")],

                                                                            ),

                                                                            ],
);

✅ Cleaner and automatically adjusts for responsiveness.

4. Writing MediaQuery Everywhere

Before: 

                                                                        
                                                                            Container(

                                                                            width: MediaQuery.of(context).size.width * 0.5,
);

After (using LayoutBuilder): 

                                                                        
                                                                            LayoutBuilder(

                                                                            builder: (context, constraints) {

                                                                            return Container(

                                                                            width: constraints.maxWidth * 0.5,

                                                                            );

                                                                            },
);

✅ Easier responsive design + avoids repeating MediaQuery .

5. Using Long Constructors for UI Widgets

Before: 

                                                                        
                                                                            Container(

                                                                            margin: EdgeInsets.only(left: 10, right: 10, top: 5, bottom: 5),

                                                                            padding: EdgeInsets.all(8),

                                                                            child: Text("Hello "),
);

After (using EdgeInsets shortcuts): 

                                                                        
                                                                            Container(

                                                                            margin: EdgeInsets.symmetric(horizontal: 10, vertical: 5),

                                                                            padding: const EdgeInsets.all(8),

                                                                            child: Text("Hello "),
);

✅ Shorter, more readable.

Conclusion

This is just beginning. There are a hundreds of shortcuts and optimizations you can give a try. Every time you replace your old code to your new optimized one. You actually move a step forward.
